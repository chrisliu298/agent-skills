# Agent Skills

My open-source skills for [Claude Code](https://docs.anthropic.com/en/docs/claude-code) and [Codex](https://openai.com/index/introducing-codex/).

## My Skills

| Skill | Description |
|-------|-------------|
| [autoresearch](https://github.com/chrisliu298/autoresearch) | Autonomous experiment loop faithful to Karpathy's autoresearch |
| [prism](https://github.com/chrisliu298/prism) | Parallel multi-agent deliberation with hard completion gate |
| [relay](https://github.com/chrisliu298/relay) | Bidirectional cross-agent relay between Claude Code and Codex |

## Install

Clone a skill into your agent's skills directory:

```bash
# Claude Code
git clone https://github.com/chrisliu298/<skill>.git ~/.claude/skills/<skill>

# Codex
git clone https://github.com/chrisliu298/<skill>.git ~/.codex/skills/<skill>
```
